# 📝 Todo List App (React)

This is a simple Todo List application built using **React**. The app allows users to manage daily tasks by adding and deleting items in an intuitive interface.

It is designed to demonstrate the use of **React functional components** and **hooks** for state management and dynamic UI updates.

---

## 🚀 Features

* Add new tasks
* Delete tasks
* Add tasks using the **Enter key**
* Real-time UI updates

---

## 🧠 Concept

The application uses React’s **useState** hook to manage:

* The list of todos
* The input field value

Tasks are stored in an array and rendered dynamically using `.map()`, making the UI responsive to every change.

---

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* CSS

---

## 🎯 Purpose

This project is a beginner-friendly implementation to understand:

* Component-based architecture
* State management in React
* Handling user input and events
